CLAUDE CODE - SIMPLE SETUP
==========================

This folder contains everything needed to set up Claude Code on a Windows computer.

WHAT'S INCLUDED:
- INSTALL-ME.bat     - Double-click this to install
- install.ps1        - PowerShell installer script
- CLAUDE.md          - Simple instructions for Claude
- settings.json      - Basic configuration

HOW TO INSTALL:
1. Copy this entire folder to the new computer
2. Double-click "INSTALL-ME.bat"
3. Click "Yes" when asked for admin permission
4. Wait for it to finish

HOW TO USE AFTER INSTALL:
1. Open Command Prompt or PowerShell
2. Type: claude
3. Press Enter
4. First time: You'll need to log in with your Anthropic account

REQUIREMENTS:
- Windows 10 or 11
- Internet connection
- Anthropic account (claude.ai)

GETTING HELP:
Just ask Claude! Type things like:
- "Help me find a file"
- "What's slowing down my computer?"
- "How do I do [anything]?"

For technical support: https://github.com/anthropics/claude-code/issues
