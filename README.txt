CLAUDE CODE - PERSONAL ASSISTANT
=================================

A friendly AI assistant that helps with everyday computer tasks.
Like having a patient tech-savvy friend available 24/7.

WHAT IT CAN DO:
- Find files and organize folders
- Install and update software
- Fix common computer problems
- Answer questions about anything
- Help with tax documents and TaxWise
- Clean up disk space
- And much more!

WHAT'S INCLUDED:
- INSTALL-ME.bat    - Double-click to install everything
- install.ps1       - The installer script
- CLAUDE.md         - Instructions that make Claude helpful
- settings.json     - Safety settings

HOW TO INSTALL:
1. Copy this folder to the computer
2. Double-click "INSTALL-ME.bat"
3. Click "Yes" when asked for permission
4. Wait for it to finish (2-3 minutes)
5. A "Claude Assistant" icon appears on desktop!

HOW TO USE:
- Double-click "Claude Assistant" on the desktop
- First time: Log in with your Anthropic account
- Then just type what you need help with!

EXAMPLE THINGS TO ASK:
- "Help me find my tax documents"
- "This computer is slow, can you help?"
- "How do I update Windows?"
- "Clean up my desktop"
- "What's using all my disk space?"

SAFETY FEATURES:
- Always asks before making changes
- Never touches system files
- Confirms before deleting anything
- Explains everything in plain English
- Blocks dangerous commands automatically

NEED HELP?
- Type "/help" to see what Claude can do
- Type "/clear" to start fresh
- Call [Your Name] if something goes wrong

REQUIREMENTS:
- Windows 10 or 11
- Internet connection
- Anthropic account (get one at claude.ai)
